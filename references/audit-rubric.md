# Audit Rubric — Deck 体检

Score out of 100, split across four dimensions. Every point deducted needs a slide number and a one-sentence reason — a score with no evidence is not a usable audit.

## 逻辑结构 (30 points)

- Does the deck have one governing message, statable in a single sentence? (−10 if the user can't state it after reading the deck once)
- Do titles read as action titles (takeaways) or topic labels? (−5 per slide with a label-only title, capped at −10)
- Does the ghost-deck test pass — titles alone tell the full story in order? (−10 if there are gaps a reader would notice)
- Is there a slide that doesn't serve the governing message at all? (−5 per orphan slide, capped at −10)

## 设计一致性 (25 points)

- One font family for headings, one for body, used consistently. (−5 if mixed without reason)
- Consistent spacing system (e.g. always 0.3" or always 0.5" gaps, not both). (−5)
- Consistent color palette tied to the topic, not a random default. (−5)
- No slide is dramatically more or less designed than its neighbors — commit to a level of polish and hold it. (−5)
- No decorative elements that exist only to look busy: accent bars, edge stripes, gratuitous icons with no informational role. (−5)

## 数据可信度 (25 points)

- Every number has a visible source or is clearly the user's own data — no unsourced statistics presented as fact. (−10 if any number looks fabricated or unattributable)
- Chart type matches the claim being made (see `chart-selection.md`) — e.g. a pie chart with 8 slices when the point is "one category dominates." (−5 per mismatched chart)
- Charts and stock imagery look like the user's actual business, not generic AI-generated placeholders (handshake photos, gear/globe icons, unrelated stock people) standing in for real content. (−5 per instance)
- Claims are appropriately hedged where the data doesn't fully support certainty — no overclaiming. (−5)

## 中文排版规范 (20 points)

Full detail in `chinese-typography.md`; the checklist below is the scored summary.

- Font pairing follows the guide (no more than 2 families, correct weight usage for hierarchy). (−5)
- Chinese/English/number mixed text has correct spacing and no orphaned single characters at line ends. (−5)
- Line spacing and font-size hierarchy are consistent across slides. (−5)
- No text overflow, no text crammed to fit, no single-character widows. (−5)
